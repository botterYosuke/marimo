# クイックスタート

Backcastを起動して、最初のノートブックを作成しましょう。

## 開発サーバーの起動

Backcastを起動するには、以下のコマンドを実行します：

```powershell
# 依存関係のインストール（初回のみ）
pnpm install

# 開発サーバーの起動（フロントエンド + バックエンド）
pnpm dev
```

または、Electronアプリとして起動：

```powershell
# 開発サーバーとElectronアプリを同時に起動
pnpm start
```

Electronアプリが起動すると、Backcastのエディタが表示されます。

## 3DモードとGridモードの切り替え

Backcastの特徴的な機能として、**3Dモード**と**Gridモード**の切り替えが可能です。

### モードの説明

- **Gridモード（vertical）**: セルをグリッドレイアウトで配置・編集する従来の表示モード
- **3Dモード（3d）**: セルを3D空間に配置し、インタラクティブに操作可能な新しい表示モード
- **デフォルトモード**: デフォルトは3Dモードです

### モードの切り替え方法

1. ツールバーの`EditViewModeSelect`コンポーネントを探します
2. ドロップダウンをクリックして、"vertical"（Gridモード）または"3d"（3Dモード）を選択します
3. 選択すると、即座に表示モードが切り替わり、セルの配置方法が変更されます

### 3Dモードの操作

3Dモードでは、以下の操作が可能です：

- **セルの移動**: セルをドラッグ&ドロップで3D空間内に移動
- **カメラ操作**: ズーム、回転、パンで3D空間を探索
- **インタラクティブな探索**: セル間の関係を3D空間で視覚的に理解

詳細は[エディタ機能の概要](../guides/editor_features/overview.md)を参照してください。

## 最初のノートブックを作成する

Backcastエディタが起動すると、新しいノートブックを作成できます。

### marimoライブラリのインポート

各Backcastノートブックの最初のセルに、以下のコードを配置することをお勧めします：

```python3
import marimo as mo
```

marimoライブラリを使用すると、BackcastノートブックでインタラクティブなUI要素、レイアウト要素、動的なmarkdownなどを使用できます。

## ノートブックをスクリプトとして実行する

Backcastノートブックは純粋なPythonファイルとして保存されるため、各ノートブックはコマンドラインからスクリプトとして実行できます：

```python
python your_notebook.py
```

また、スクリプトに[CLI引数を渡す](../guides/scripts.md)こともできます。
