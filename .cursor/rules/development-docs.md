# 開発ドキュメント参照ルール

## 概要

開発時には、`marimo-main/development_docs` フォルダ内のドキュメントを参照すること。

## 利用可能なドキュメント

`development_docs` フォルダには以下の開発関連ドキュメントが含まれています：

- **testing.md**: テストに関する情報
- **openapi.md**: OpenAPIに関する情報
- **prompts.md**: プロンプトに関する情報
- **pyodide.md**: Pyodideに関する情報
- **traces.md**: トレースに関する情報
- **adding_backend_and_mcp_tools.md**: バックエンドとMCPツールの追加方法
- **adding_lint_rules.md**: リントルールの追加方法

## 使用方法

開発中に特定のトピックに関する情報が必要な場合は、まず `development_docs` フォルダ内の該当ドキュメントを確認すること。

これらのドキュメントには、プロジェクト固有の開発ガイドライン、ベストプラクティス、手順などが記載されています。
