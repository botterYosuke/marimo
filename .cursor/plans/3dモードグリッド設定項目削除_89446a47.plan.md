# 3Dモ

ードグリッド設定項目削除

## 概要

3Dモードのグリッド設定から以下の5項目を削除し、2Dモードと同様に固定値として扱う：

- Grid Spacing X（X方向の間隔）
- Grid Depth（Z方向の間隔、`spacingZ`）
- Grid Height（Y方向の高さ、`spacingY`）
- Grid Opacity（グリッド平面の透明度、0-1）
- Snap to Grid（グリッドにスナップするか）

## 変更ファイル

### 1. [src/components/editor/renderers/3d-layout/types.ts](src/components/editor/renderers/3d-layout/types.ts)

- **実装済み**: `Grid3DConfig`インターフェースから`spacingX`, `spacingY`, `spacingZ`, `gridOpacity`, `snapToGrid`は既に削除されています。`DEFAULT_GRID_3D_CONFIG`からもこれらのデフォルト値は削除済みです。

### 2. [src/components/editor/renderers/3d-layout/grid-3d-controls.tsx](src/components/editor/renderers/3d-layout/grid-3d-controls.tsx)

- **実装済み**: UIから以下の5つのコントロール要素は既に削除されています：
- Grid Spacing X
- Grid Depth
