# ユーザードキュメント参照ルール

## 概要

marimoを使うタスクを実行する際は、`marimo-main/docs` フォルダ内のドキュメントを参照すること。

## 利用可能なドキュメント

`docs` フォルダには以下のユーザー向けドキュメントが含まれています：

- **index.md**: marimoの概要、インストール方法、主な機能
- **getting_started/**: 初心者向けチュートリアル、クイックスタート、主要概念
- **guides/**: 使い方ガイド（インタラクティブ性、データ操作、デプロイ、設定など）
- **api/**: APIリファレンス
- **faq.md**: よくある質問
- **troubleshooting.md**: トラブルシューティング
- **examples/**: 使用例
- **recipes.md**: レシピ集

## 使用方法

marimoの使用方法、機能、設定、デプロイ方法などに関する質問やタスクを実行する際は、まず `docs` フォルダ内の該当ドキュメントを確認すること。

これらのドキュメントには、marimoの使い方、ベストプラクティス、実装例などが記載されています。
